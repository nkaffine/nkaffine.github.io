StackRanked

Real-time group ranking game

StackRanked is a live multiplayer game where groups create shared rankings by voting head to head in real time.

Privacy Policy

Last updated: February 15, 2026

1. Data We Collect

StackRanked collects limited data required to provide app functionality:

  • Anonymous User ID
  • Username chosen by the user
  • Gameplay data including votes, rankings, and group participation

2. How Data Is Used

Data is used solely to:

  • Enable real-time group functionality
  • Save gameplay progress
  • Maintain rankings
  • Authenticate anonymous sessions

3. Data Sharing

We do not:

  • Sell personal data
  • Use third-party advertising
  • Track users across apps or websites
  • Share data with data brokers

Data is stored using Supabase to provide core app functionality.

4. Account & Data Deletion

Users may request deletion of their account and associated gameplay data by contacting:

stackranked.game@gmail.com

In-app deletion will be available in a future update.

5. Contact

For privacy-related inquiries:
